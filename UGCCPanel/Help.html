Overview
This web based control panel allows you to easily manage your game servers or other programs from anywhere.  If you are unsure about something; all items have tool tips that will display if you hold your mouse cursor over them.

Main Menu
The first screen you see after you login is the server list.  This screen lists all the items you have permissions to access.  From this screen you can:
  • Start/Stop the server-- To do this click the status text to toggle the status.  For example, if it says 'Down' clicking it will start it.  Clicking it when it says 'Up' will stop it.
  • Goto the server management page--You can click the 'Server Name' text to take you to the server's management page where more options are available for the server.
  • My Account Page.  You can change your password and view what email address we have on record.
  • View the support contact information
  • Create and manage support tickets if enabled
  • View the news
  • Log off

Server Management Page
  • Start/Stop/Restart the server
  • Use the web RCON tool if supported for your server
  • Refresh the server status
  • View server detailsetails
  • View user notes--allows you to keep your own private notes
  • Query the server--allows you see basic server info and check to see if players are playing
  • Edit the server's command line (if allowed)--edit the command line arguments used to start your server
  • Use the file tool (if allowed)
    • Use the Update tool (if configured)
  • Schedule timed restarts (if configured)--allows you to set the time and day that your server is automatically stopped and then restarted
  • Srvr Image - create images that you can use on other sites to show current status of your game server.
  • Srvr Variables - edit any variables made available to you
  • Use any extras features/programs setup by the administrator

File Toole Tool
The file tool allows you to:ou to:
  • Upload files
  • Rename files
  • Delete files
  • Create files
  • Edit files
  • Create folders
  • Delete folders
  • Decompress archives (7z, ZIP, GZIP, BZIP2, TAR, RAR, CAB, ISO, ARJ, LZH, CHM, MSI, WIM, Z, CPIO, RPM, DEB and NSIS )

Update Tool
If you are running a server that can be updated, and the administrator has allowed you to update your server, you can do so with this tool.  Click the begin button to start the update.  The page updates automatically every 120 secods, and your force a manual update by clicking the refresh button.. If you close the browser window you will not be able to further monitor the update, but it will continue to run.  You can navigate away from the page and come back to it later to check it's status as long as your session doesn't timeout.

Getting Help
If you need more help, please contact the administrator.  You can submit a support ticket or contact us via the methods listed on the Support Info page.
